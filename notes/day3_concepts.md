# Day 3 Concepts

## 1. EEG 是什么
EEG 是通过在头皮表面放置电极，记录一段时间内脑活动引起的电位变化。它反映的是大量神经元群体活动在头皮上形成的信号。EEG 的优点是时间分辨率高、非侵入性强、设备相对便携，因此常用于脑机接口和认知任务研究。它的缺点是噪声较多，空间分辨率有限。

## 2. Motor Imagery 是什么
Motor imagery 指的是在不真正执行动作的情况下，在脑中想象某个动作的过程，例如想象抬手或抬腿。虽然没有真实动作输出，但脑活动中仍然会出现与运动相关的模式变化。因为任务定义清楚、实验范式成熟，所以 motor imagery 是 EEG 解码和 BCI 中很常见的任务。

## 3. Decoding 是什么
在这个项目中，decoding 指的是根据 EEG 信号去预测被试当前的动作意图或任务类别。输入是一定时间窗口内的 EEG 数据，输出是动作对应的标签，例如左手、右手或其他任务类别。对于当前这个项目来说，本质上是一个有监督的分类问题。

## 4. Event 是什么
Event 指的是 EEG 连续时间序列中的事件标记，用来表示某个实验任务、刺激或行为在什么时间点发生。例如提示被试开始想象某个动作的信号出现时，就可以记录为一个 event。它的作用是帮助我们从连续信号中定位与任务相关的时间点，并据此切出后续分析所需的样本片段。

## 5. Epoch 是什么
Raw signal 是电极记录到的连续 EEG 原始信号。Epoch 是围绕某个 event 从连续信号中切出来的一小段时间窗，它对应一个具体任务或实验条件。训练模型时通常使用 epoch，而不是整段 raw signal，因为 epoch 更容易和标签一一对应。Epoch 不一定代表已经完成全部处理，但它通常是后续分析和建模的基本样本单位。

## 6. 我的当前理解
我现在做的是一个 EEG motor imagery decoding 项目。目标是从连续脑电信号中找到与任务相关的时间片段，并把这些片段整理成可用于建模的样本。之后我会对这些数据进行预处理、特征提取和分类建模，最终实现根据 EEG 预测动作意图的任务。这个项目本质上是把脑信号分析和机器学习分类结合起来的一个入门脑科学工程项目。
