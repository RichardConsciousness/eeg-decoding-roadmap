# Day 5 Dataset and Labels

## 1. 我今天用的是哪个数据集、哪个 subject、哪个 run
目前使用的是 EEGBCI 数据集中的 subject 1、run 6 数据。

## 2. event_id 里有哪些标签
当前 event_id 中包含 3 个标签，分别是 T0、T1、T2。

## 3. epochs 的 shape 是什么
当前 epochs 的 shape 是 (29, 64, 801)，表示共有 29 个样本，每个样本有 64 个 EEG 通道，每个通道包含 801 个时间点。

## 4. 每类样本数量如何
T0、T1、T2 的样本数分别为 14、7、8。当前类别分布不完全平衡，T0 明显多于另外两类。

## 5. 我当前面对的是一个什么分类任务
当前任务可以看作一个三分类监督学习问题。每个样本是一个 shape 为 64×801 的 EEG epoch，对应一个标签 T0、T1 或 T2。下一步需要进一步确认这些标签在 run 6 中分别对应什么具体任务语义。
