# EEG Decoding Roadmap

## 1. 项目背景
这个仓库用于记录我的第一个脑电（EEG）解码项目，以及整个跨学科学习过程。

## 2. 当前目标
在 60 天内完成一个可复现的 EEG 运动想象（motor imagery）解码项目，形成可展示的 GitHub 仓库、实验记录与项目总结。

## 3. 当前主项目
主项目：EEG motor imagery decoding

计划完成的内容包括：
- 理解 EEG 数据结构与基本概念
- 使用 MNE 进行数据读取与预处理
- 构建基础分类流程
- 完成至少一个传统机器学习 baseline
- 完成至少一个深度学习 baseline
- 输出结果图、项目总结

## 4. 工具链
- Python
- Jupyter
- MNE-Python
- scikit-learn
- PyTorch
- Braindecode
- MOABB

## 5. 仓库结构
- `notes/`：概念笔记与学习记录
- `notebooks/`：实验 notebook
- `src/`：后续整理出来的可复用代码
- `figures/`：图表与可视化结果
- `reports/`：阶段总结与项目报告
